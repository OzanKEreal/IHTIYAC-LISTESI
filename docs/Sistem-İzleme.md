### SİSTEM İZLEME VE PERFORMANS ARAÇLARI
---

## Sistem İzleme

### 1. Task Manager (Windows)
- **Platform**: Windows
- **Fiyat**: Ücretsiz (Windows ile birlikte)
- **Özellikler**:
  - Process monitoring
  - CPU, Memory, Disk, Network usage
  - Performance graphs
  - Startup apps
  - Services
  - Process details
  - End task
- **En İyi Kullanım**: Windows temel sistem izleme
- **Web**: Built-in to Windows

### 2. Activity Monitor (macOS)
- **Platform**: macOS
- **Fiyat**: Ücretsiz (macOS ile birlikte)
- **Özellikler**:
  - CPU, Memory, Energy, Disk, Network
  - Process monitoring
  - Force quit
  - View detailed process info
  - Energy impact
  - Memory pressure
  - Disk activity
- **En İyi Kullanım**: Mac temel sistem izleme
- **Web**: Built-in to macOS

### 3. htop
- **Platform**: Linux, macOS (via Homebrew), Windows (via WSL)
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Interactive process viewer
  - CPU, Memory usage
  - Process management
  - Process tree
  - Kill processes
  - Color-coded
  - Customizable
- **En İyi Kullanım**: Linux terminal sistem izleme
- **Web**: https://htop.dev

### 4. Process Explorer
- **Platform**: Windows
- **Fiyat**: Ücretsiz
- **Özellikler**:
  - Advanced process viewer
  - Process tree
  - Handles & DLLs
  - CPU usage
  - Memory usage
  - Kill process
  - Search
  - System info
- **En İyi Kullanım**: Windows detaylı process monitoring
- **Web**: https://learn.microsoft.com/sysinternals/downloads/process-explorer

### 5. Resource Monitor
- **Platform**: Windows
- **Fiyat**: Ücretsiz (Windows ile birlikte)
- **Özellikler**:
  - CPU, Memory, Disk, Network detailed monitoring
  - Processes with handles
  - Disk activity
  - Network connections
  - File handles
  - Real-time graphs
- **En İyi Kullanım**: Windows kaynak detayları
- **Web**: Built-in to Windows

## Performance Monitoring

### 11. PerfMon (Windows Performance Monitor)
- **Platform**: Windows
- **Fiyat**: Ücretsiz (Windows ile birlikte)
- **Özellikler**:
  - Performance counters
  - Real-time graphs
  - Data collector sets
  - Reports
  - Custom counters
  - Alerts
  - Long-term monitoring
- **En İyi Kullanım**: Windows performance data
- **Web**: Built-in to Windows

### 12. Glances
- **Platform**: Linux, macOS, Windows (via Python)
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Open source
  - System monitoring
  - CPU, Memory, Disk, Network
  - Processes
  - Web interface
  - Export (CSV, JSON)
  - Alert thresholds
- **En İyi Kullanım**: Linux monitoring tool
- **Web**: https://nicolargo.github.io/glances

### 13. nmon (Nigel's Monitor)
- **Platform**: Linux, AIX
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - System monitoring
  - CPU, Memory, Disk, Network
  - Processes
  - NFS
  - Top processes
  - Save data
  - Curses UI
- **En İyi Kullanım**: Linux & AIX monitoring
- **Web**: https://sourceforge.net/projects/nmon

### 14. Netdata
- **Platform**: Linux, macOS, FreeBSD, Windows
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:

  - 1000+ metrics
  - Web dashboard
  - CPU, Memory, Disk, Network

  - Alerts
  - Plugins
- **En İyi Kullanım**: Real-time dashboard
- **Web**: https://www.netdata.cloud

### 15. Prometheus
- **Platform**: Linux, macOS, Windows (via binary)
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:

- **En İyi Kullanım**: Monitoring & alerting system
- **Web**: https://prometheus.io
