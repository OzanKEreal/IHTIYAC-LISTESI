## Markdown Editörleri

### 1. Typora
- **Platform**: Windows, macOS, Linux
- **Fiyat**: Ücretsiz (beta) / Ücretli ($14.99 one-time)
- **Özellikler**:
  - What You See Is What You Mean (WYSIWYM)
  - Live preview
  - Focus mode
  - Typewriter mode
  - Image support
  - Export (HTML, PDF, Word, etc.)
  - Tables
  - MathJax (LaTeX)
- **En İyi Kullanım**: Seamless markdown editing
- **Web**: https://typora.io
