## Ekip İletişimi ve Collaboration

### 1. Slack
- **Platform**: Web, Windows, macOS, iOS, Android
- **Fiyat**: Ücretsiz (sınırlı) / Ücretli ($7.25-15/ay)
- **Özellikler**:
  - Channels & DMs
  - File sharing
  - Search
  - Integrations (1000+)
  - Huddles (audio/video)
  - Workflow Builder
  - Canvas
  - Clips (video mesajlar)
  - AI features
- **En İyi Kullanım**: Team communication
- **Web**: https://slack.com
