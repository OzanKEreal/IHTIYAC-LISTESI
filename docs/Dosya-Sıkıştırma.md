## Popüler Arşiv Programları

### 1. WinRAR
- **Platform**: Windows
- **Fiyat**: Ücretsiz deneme / Ücretli ($29 one-time)
- **Özellikler**:
  - RAR, ZIP, 7Z, CAB, ARJ, LZH, TAR, GZ, ACE, UUE, BZ2, JAR, ISO, Z, 7-Zip
  - Şifre koruması
  - Recovery record
  - Solid arşivleme
  - Multi-volume
  - Self-extracting arşivler
  - Integration (Windows)
- **En İyi Kullanım**: Windows için standart arşiv programı
- **Web**: https://www.win-rar.com

### 2. 7-Zip
- **Platform**: Windows, Linux
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Open source

  - Integration (Windows)
  - Command line
  - Self-extracting
  - Multi-language
- **En İyi Kullanım**: Ücretsiz open-source alternatif
- **Web**: https://www.7-zip.org

## Command Line Tools

### 16. tar (Unix)
- **Platform**: Linux, macOS, Windows (via WSL)
- **Fiyat**: Ücretsiz
- **Özellikler**:
  - Open source
  - Standard Unix archive
  - Compression (gzip, bzip2, xz)
  - Preserves permissions
  - Incremental backups
  - Widely used
- **En İyi Kullanım**: Unix-style arşivleme
- **Web**: https://www.gnu.org/software/tar

### 17. gzip
- **Platform**: Linux, macOS, Windows
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Standard compression
  - Fast
  - Widely compatible
  - Command line
  - Decompression
- **En İyi Kullanım**: Gzip dosyaları
- **Web**: https://www.gnu.org/software/gzip

### 18. bzip2
- **Platform**: Linux, macOS, Windows
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - High compression
  - Slower than gzip
  - Good for text files
  - Command line
  - Decompression
- **En İyi Kullanım**: Yüksek sıkıştırma ihtiyacı
- **Web**: https://sourceware.org/bzip2

### 19. xz
- **Platform**: Linux, macOS, Windows
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - LZMA compression
  - Very high compression
  - Slow compression, fast decompression
  - Command line
  - Modern standard
- **En İyi Kullanım**: Yüksek sıkıştırma ihtiyacı
- **Web**: https://tukaani.org/xz

### 20. zstd (Zstandard)
- **Platform**: Linux, macOS, Windows
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Fast compression & decompression
  - Good compression ratio
  - Modern standard
  - Command line
  - Wide adoption
- **En İyi Kullanım**: Hızlı ve verimli sıkıştırma
- **Web**: https://github.com/facebook/zstd
