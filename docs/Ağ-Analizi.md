## Network Analyzer ve Scanner'lar

### 1. Wireshark
- **Platform**: Windows, macOS, Linux
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Deep packet inspection
  - Protocol analyzer
  - Live capture
  - Read/write many formats (pcap, pcapng, etc.)
  - Filters & display filters
  - Decryption (SSL/TLS)
  - Packet coloring
  - Export objects
  - Statistics & graphs
- **En İyi Kullanım**: Deep packet analysis
- **Web**: https://www.wireshark.org

### 2. Nmap (Network Mapper)
- **Platform**: Windows, macOS, Linux
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Network discovery
  - Port scanning
  - Service detection
  - OS detection
  - NSE (Nmap Scripting Engine)
  - Output formats (XML, JSON, etc.)
  - Firewall evasion
- **En İyi Kullanım**: Network scanning & discovery
- **Web**: https://nmap.org

### 3. Fing
- **Platform**: Windows, macOS, iOS, Android
- **Fiyat**: Ücretsiz / Ücretli (Desktop Pro - $29.99)
- **Özellikler**:
  - Network scanner
  - Device discovery
  - Port scanning
  - IP address lookup
  - Network speed test
  - Wake-on-LAN
  - Security analysis
- **En İyi Kullanım**: Quick network scanner
- **Web**: https://www.fing.com

### 4. Advanced IP Scanner
- **Platform**: Windows
- **Fiyat**: Ücretsiz
- **Özellikler**:
  - Network scanning
  - IP range scanning
  - MAC address detection
  - Remote control
  - Wake-on-LAN
  - Share folders
  - HTTP, HTTPS, RDP access
- **En İyi Kullanım**: Windows network scanner
- **Web**: https://www.advanced-ip-scanner.com

### 5. Angry IP Scanner
- **Platform**: Windows, macOS, Linux
- **Fiyat**: Ücretsiz ve Açık Kaynak
- **Özellikler**:
  - Open source
  - IP range scanning
  - Port scanning
  - Ping
  - NetBIOS
  - Fetchers
  - Export results (CSV, TXT, etc.)
  - Extensible via plugins
- **En İyi Kullanım**: Cross-platform IP scanner
- **Web**: https://angryip.org
